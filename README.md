# 🎮 Hangman Game in Python

A simple console-based Hangman Game developed using Python. This project was created to practice basic Python programming concepts such as loops, conditions, strings, lists, and the random module.

---

## 📌 Features

- Random word selection
- 5 predefined words
- Maximum 6 incorrect guesses
- Displays guessed letters
- User-friendly console interface
- Beginner-friendly Python project

---

## 🛠 Technologies Used

- Python 3
- VS Code

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project in VS Code.
3. Run the following command:

```bash
python hangman.py
```

## 📖 Concepts Used

- Random Module

---

## 🎯 Game Rules

- A random word is selected.
- Guess one letter at a time.
- You have a maximum of **6 wrong guesses**.
- Guess the complete word before your chances end.
